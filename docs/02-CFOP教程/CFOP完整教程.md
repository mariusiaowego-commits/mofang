# CFOP 完整速拧教程

> [!success] 前言
> CFOP (Cross → F2L → OLL → PLL) 是世界上最主流的速拧方法，几乎所有世界纪录保持者都使用这个方法。掌握后sub15秒甚至sub10秒都是可能的。

---

## CFOP 公式总数

| 步骤 | 情况数 | 学习优先级 |
|------|--------|-----------|
| **Cross** | 无数种（无固定公式） | 最高 |
| **F2L** | 41种标准情况 | 高 |
| **OLL** | 57种情况 | 中 |
| **PLL** | 21种情况 | 高 |

> [!note] 总计：119个标准公式

## 各阶段详解

---

### 一、Cross (十字)

> [!important] 核心原则
> **永远在底面做十字！** 这是和层先法最大的区别，也是提速的关键。

#### 目标：8步内完成，观察时间内规划好

#### 核心原则
1. 永远在底面做十字（不用翻面）
2. 4个棱块都要对齐侧面中心
3. 8步定律：95%的情况8步内可以完成

#### 训练方法

> [!example] 十字专项训练
> 1. 只练十字，每天100个
> 2. 不计时，只看步数
> 3. 15秒观察，想好整个十字的4个棱块顺序
> 4. 拧的时候不看十字，只按计划拧

> [!tip] 十字判断练习
> 给一个打乱，15秒观察，不拧魔方，说出4个棱块的顺序，然后拧出来验证

---

### 二、F2L (First Two Layers)

> [!important] 核心思想
> **角棱配对 → 一起入槽**
> F2L不需要死记硬背，理解4种基本情况后可以推导所有情况！

#### F2L 4种基本情况

> [!example] 情况1：角在顶层，棱在顶层，白朝上
> ```
> R U R'
> ```

> [!example] 情况2：角在顶层，棱在顶层，白朝前
> ```
> y' U' R' U R
> ```

> [!tip] 藏角技巧
> ```
> R U' R' （把角藏到底层）
> ```

> [!tip] 藏棱技巧
> ```
> F' U F （把棱藏到底层）
> ```

#### F2L 学习顺序

| 批次 | 内容 | 目标 |
|------|------|------|
| **第一批** | 4种基本情况 + 角棱已配对 + 槽位被占 | 必须先掌握 |
| **第二批** | 角在顶层棱在中层 + 角在底层棱在顶层 + 镜像 | 常见情况 |
| **第三批** | 所有剩余情况 + 非标 + 多向 | 精通 |

---

### 三、OLL (Orientation of Last Layer)

> [!goal] 目标
> 一次公式让顶面全部同色

#### 第一阶段：7个十字OLL（必须先学）

> [!example] OLL 21 (小鱼1 / Sune)
> ```
> R U R' U R U2 R'
> ```

> [!example] OLL 22 (小鱼2 / Anti-Sune)
> ```
> L' U' L U' L' U2 L
> ```

> [!example] OLL 23
> ```
> R U2 R' U' R U' R'
> ```

> [!example] OLL 24
> ```
> L' U2 L U L' U L
> ```

> [!example] OLL 25
> ```
> R U R' U R U' R' U R U2 R'
> ```

> [!example] OLL 26 (双角对边)
> ```
> f (R U R' U') (R U R' U') f'
> ```

> [!example] OLL 27 (十字)
> ```
> F (R U R' U') F'
> ```

#### 第二阶段：全部57个OLL

按优先级学习：
1. 点系列（7个）
2. 线系列
3. L系列
4. 十字系列
5. 其他

---

### 四、PLL (Permutation of Last Layer)

> [!goal] 目标
> 还原最后一层所有块的位置

> [!important] 强烈建议：PLL全部21个先学完！
> 这是投入产出比最高的阶段！

#### 最常用的6个PLL（先学这些）

> [!example] PLL 1 - Ua Perm（三棱顺换）
> ```
> M2 U M U2 M' U M2
> ```

> [!example] PLL 2 - Ub Perm（三棱逆换）
> ```
> M2 U' M U2 M' U' M2
> ```

> [!example] PLL 3 - Z Perm（对棱换）
> ```
> M2 U M2 U M' U2 M2
> ```

> [!example] PLL 4 - H Perm（十字对换）
> ```
> M2 U' M2 U2 M2 U' M2
> ```

> [!example] PLL 5 - Aa Perm
> ```
> x' R U R' D R U' R' D' R U R' D R U' R' D' x
> ```

> [!example] PLL 6 - Ab Perm
> ```
> x R' U R' D2 R U' R' D2 R2 x'
> ```

#### 完整21个PLL学习顺序

按类型分组学习：
1. ✅ 棱换系列（U, H, Z）
2. ✅ 角换系列（A, E）
3. ⏳ 角棱换系列（G, R, J, T, Y, V, N, F）

## 🧠 公式记忆技巧

### 肌肉记忆法

> [!example] 标准流程
> 1. 慢速做5遍，注意每一步
> 2. 中速做10遍，形成连贯
> 3. 快速做20遍，肌肉记忆形成
> 4. 每小时复习5遍，每天复习

### 分组记忆

> [!tip] 分组原则
> - 按形状相似性分组
> - 每组3-5个公式
> - 掌握一组再下一组
> - 旧公式每天必复习

### 间隔重复法

> [!success] 艾宾浩斯遗忘曲线应用
> - 第1天：学习新公式
> - 第2天：复习第1天的
> - 第4天：再复习
> - 第7天：再复习
> - 第14天：再复习

## ❓ 常见问题

> [!faq] 公式太多记不住怎么办？
> CFOP不是一次性学完所有公式的！按顺序来，先学最常用的，边用边记。PLL 21个每天1个，3周就搞定了！

> [!faq] F2L一定要背吗？
> 不用！理解原理后可以自己推导，背是为了更快。推荐先理解，再选择性背高频情况。

> [!faq] 为什么我学了CFOP反而变慢了？
> 正常！新方法需要适应期，坚持2-3周就会超过原来的速度。这是"J型曲线"，先降后升！

> [!faq] 什么时候开始学OLL？
> PLL学完，F2L比较熟练了（30-40秒水平）再开始学OLL。
