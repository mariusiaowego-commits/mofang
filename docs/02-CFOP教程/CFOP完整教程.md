# CFOP 完整教程

CFOP (Cross → F2L → OLL → PLL) 是世界上最主流的速拧方法，几乎所有世界纪录保持者都使用这个方法。

## CFOP 公式总数

- **Cross**: 无数种情况（无固定公式）
- **F2L**: 41种标准情况
- **OLL**: 57种情况
- **PLL**: 21种情况

**总计：119个公式**

---

## 学习顺序（推荐）

### 第一阶段：基础CFOP（目标：45s）
1. ✅ Cross（底面十字）
2. ✅ F2L 基础理解 + 常见10种
3. ✅ 7个十字OLL
4. ✅ 21个PLL

### 第二阶段：完整CFOP（目标：25s）
5. F2L 全部41种
6. OLL 全部57种
7. 观察技巧训练

### 第三阶段：高级优化（目标：15s）
8. F2L 非标
9. OLL/PLL 多向公式
10. 预判和衔接

## 各阶段详解

### 一、Cross (十字)

**目标：8步内完成，观察时间内规划好**

**核心原则**：
1. 永远在底面做十字（不用翻面）
2. 4个棱块都要对齐侧面中心
3. 8步定律：95%的情况8步内可以完成

**训练方法**：
- 只练十字，每天100个
- 不计时，只看步数
- 15秒观察，想好整个十字

**十字判断练习**：
- 给一个打乱，15秒观察
- 不拧魔方，说出4个棱块的顺序
- 然后拧出来验证

---

### 二、F2L (First Two Layers)

**核心思想：角棱配对 → 一起入槽**

F2L不需要死记硬背，理解4种基本情况后可以推导所有情况。

#### F2L 4种基本情况

**情况1：角在顶层，棱在顶层，白朝上**
```
U R U' R'
```

**情况2：角在顶层，棱在顶层，白朝前**
```
y' U' R' U R
```

**情况3：藏角技巧**
```
R U' R' （把角藏到底层）
```

**情况4：藏棱技巧**
```
F' U F （把棱藏到底层）
```

#### F2L 学习顺序

**第一批（必须先掌握）**：
1. 4种基本情况
2. 角棱已配对的情况
3. 槽位被占的情况

**第二批（常见情况）**：
4. 角在顶层，棱在中层
5. 角在底层，棱在顶层
6. 镜像情况

**第三批（全部41种）**：
7. 所有剩余情况
8. 非标情况
9. 多向F2L

---

### 三、OLL (Orientation of Last Layer)

**目标：一次公式让顶面全部同色**

#### 第一阶段：7个十字OLL

当你已经做出顶面十字时，只有7种情况：

**OLL 21 (小鱼1)**
```
R U R' U R U2 R'
```

**OLL 22 (小鱼2)**
```
L' U' L U' L' U2 L
```

**OLL 23 (Sune反)**
```
R U2 R' U' R U' R'
```

**OLL 24 (Anti-Sune)**
```
L' U2 L U L' U L
```

**OLL 25 (点+线)**
```
R U R' U R U' R' U R U2 R'
```

**OLL 26 (双角对边)**
```
f (R U R' U') (R U R' U') f'
```

**OLL 27 (十字)**
```
F (R U R' U') F'
```

#### 第二阶段：全部57个OLL

按优先级学习：
1. 点系列（1-7）
2. 线系列（8-13）
3. L系列（14-20）
4. 十字系列（21-27）
5. 其他

---

### 四、PLL (Permutation of Last Layer)

**目标：还原最后一层所有块的位置**

21个PLL，强烈建议全部学会！

#### 最常用的6个PLL（先学这些）

**PLL 1 - Ua Perm（三棱顺换）**
```
M2 U M U2 M' U M2
```

**PLL 2 - Ub Perm（三棱逆换）**
```
M2 U' M U2 M' U' M2
```

**PLL 3 - Z Perm（对棱换）**
```
M2 U M2 U M' U2 M2
```

**PLL 4 - H Perm（十字对换）**
```
M2 U' M2 U2 M2 U' M2
```

**PLL 5 - Aa Perm**
```
x' R U R' D R U' R' D' R U R' D R U' R' D' x
```

**PLL 6 - Ab Perm**
```
x R' U R' D2 R U' R' D2 R2 x'
```

#### 完整21个PLL

按类型分组学习：
1. 角换系列（A, E, F）
2. 棱换系列（U, H, Z）
3. 角棱换系列（G, R, J, T, Y, V, N）

## 公式记忆技巧

### 肌肉记忆法
1. 慢速做5遍，注意每一步
2. 中速做10遍，形成连贯
3. 快速做20遍，肌肉记忆形成
4. 每小时复习5遍，每天复习

### 分组记忆
- 按形状相似性分组
- 每组3-5个公式
- 掌握一组再下一组
- 旧公式每天复习

### 间隔重复
- 第1天：学习新公式
- 第2天：复习第1天的
- 第4天：再复习
- 第7天：再复习
- 第14天：再复习

## 常见问题

**Q: 公式太多记不住怎么办？**
A: CFOP不是一次性学完所有公式的，按顺序来，先学最常用的，边用边记。

**Q: F2L一定要背吗？**
A: 不用，理解原理后可以自己推导，背是为了更快。

**Q: 为什么我学了CFOP反而变慢了？**
A: 正常，新方法需要适应期，坚持2-3周就会超过原来的速度。

**Q: 什么时候开始学OLL？**
A: PLL学完，F2L比较熟练了（30-40s水平）再开始学OLL。
