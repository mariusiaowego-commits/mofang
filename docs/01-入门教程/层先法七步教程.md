# 三阶魔方入门教程 - 层先法

> [!success] 前言
> 层先法（Layer by Layer）是最适合初学者的魔方还原方法，简单易懂，步骤清晰。掌握后可以在2分钟内还原魔方，是学习高级CFOP方法的基础。

![[00-七步概览.png|600]]

---

## 魔方基础

### 魔方结构

![[00-魔方结构.png|500]]

- **中心块**：6个，固定不动，决定颜色
- **棱块**：12个，每种2个颜色
- **角块**：8个，每种3个颜色

### 标准配色

| 对面颜色 | 说明 |
|----------|------|
| 白 ↔ 黄 |  |
| 红 ↔ 橙 |  |
| 蓝 ↔ 绿 |  |

> [!tip] 中国标准
> **白底黄顶，红前蓝右** - 记住这个标准姿势！

### 转动符号

| 符号 | 说明 | 符号 | 说明 |
|------|------|------|------|
| R | 右面顺时针 | R' | 右面逆时针 |
| L | 左面顺时针 | L' | 左面逆时针 |
| U | 顶面顺时针 | U' | 顶面逆时针 |
| D | 底面顺时针 | D' | 底面逆时针 |
| F | 前面顺时针 | F' | 前面逆时针 |
| B | 后面顺时针 | B' | 后面逆时针 |
| x | 整体R方向转 | y | 整体U方向转 |
| z | 整体F方向转 |  |  |

> [!note] 加2表示转动180度
> 如 `R2` = 右面转180度，顺逆都一样

## 七步层先法

---

### 第一步：底面十字（白色）

> [!goal] 目标
> 在白色底面做出完美十字，4个棱块都对齐侧面中心颜色

![[01-底面十字.png|400]]

**要点**：
1. 找到白色中心块（通常作为底面）
2. 逐个找到带白色的棱块
3. 将它们转到正确位置，对齐侧面中心
4. **十字要求：4个棱块都对齐侧面中心颜色**

> [!example] 常见手法
> - `F U R U'`

---

### 第二步：底面角块

> [!goal] 目标
> 还原4个白色角块，完成第一层

**通用公式（角块在顶层右前）**：
```
R U R'
```

**角块在底层但位置错误**：
```
R U R' （先弄到顶层，再用通用公式）
```

---

### 第三步：第二层棱块

> [!goal] 目标
> 还原中间层的4个棱块

**情况1：棱块要往左去**：
```
U' L' U L U F U' F'
```

**情况2：棱块要往右去**：
```
U R U' R' U' F' U F
```

> [!tip] 棱块在错误位置
> 先用上面公式把它弄出去，再重新放进去

---

### 第四步：顶面十字（黄色）

> [!goal] 目标
> 在黄色顶面做出十字

![[04-顶面十字三状态.png|500]]

**三种情况，同一公式，反复做**：
```
F R U R' U' F'
```

| 情况 | 形状 | 需要次数 |
|------|------|----------|
| 1 | 一个点 | 3次 |
| 2 | 一条线（横向） | 2次 |
| 3 | L型（直角） | 1次 |

---

### 第五步：顶面翻色

> [!goal] 目标
> 让顶面全部变成黄色

![[05-小鱼图案.png|500]]

**小鱼公式（7种情况通用）**：

**小鱼1（黄色在左后）**：
```
R U R' U R U2 R'
```

**小鱼2（黄色在右前）**：
```
L' U' L U' L' U2 L
```

> [!tip] 其他情况处理
> 随便做一个小鱼公式，变成小鱼情况再做

---

### 第六步：顶角归位

> [!goal] 目标
> 让四个角块到正确位置

![[06-眼睛图案.png|400]]

**找眼睛**：找到两个角块侧面颜色相同的（像眼睛）

**公式**：
```
R B' R F2 R' B R F2 R2
```

> [!example] 用法
> - 把"眼睛"放在背面（远离你）
> - 做公式
> - 如果没有眼睛，随便做一次公式就有了

---

### 第七步：顶棱归位

> [!goal] 目标
> 最后一步，还原整个魔方！

**情况1：一个面已好，三个棱块要循环**

**三棱顺换**：
```
(R U' R) (U R U R) (U' R' U' R2)
```

**三棱逆换**：
```
(R2' U R' U') (R' U' R' U R)
```

> [!tip] 情况2：四个棱块都不对
> 先做一遍上面的公式，变成情况1

## 恭喜！

> [!success] 🎉 你已经学会了三阶魔方的基础还原方法！

### 下一步建议

1. **熟练层先法**：反复练习，直到能在2分钟内完成
2. **学习手法**：开始练习正确的手指发力
3. **优化十字**：学习在底面做十字（不用翻魔方）
4. **进入CFOP**：当你稳定在1分钟左右时，开始学习[[CFOP完整教程]]高级方法

### 练习建议

| 阶段 | 目标时间 | 建议 |
|------|----------|------|
| 刚学会 | 5分钟 | 记住所有公式 |
| 熟练 | 3分钟 | 手法优化 |
| 进阶 | 2分钟 | 观察速度提升 |
| 瓶颈 | 1分钟 | 准备学习CFOP |
