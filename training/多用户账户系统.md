# 多用户账户系统

## 用户数据结构

```yaml
users:
  user_id:
    profile:
      name: 用户名
      start_date: 开始日期
      level: 当前水平
      goal: 目标
    training:
      total_solves: 总还原次数
      total_hours: 总训练时长
      best_single: 最佳单次
      best_ao5: 最佳AO5
      best_ao12: 最佳AO12
    progress:
      formulas_learned:
        f2l: []
        oll: []
        pll: []
      milestones: []
    records:
      - date: 日期
        session: 训练记录
```

---

## 用户注册模板

保存位置：`training/users/[用户名].yaml`

```yaml
# 用户配置文件
profile:
  name: 
  start_date: YYYY-MM-DD
  current_level: 入门/进阶/熟练/高手/顶级
  target_level: 目标水平
  daily_goal: 每日目标次数
  weekly_goal: 每周目标小时数

knowledge:
  method: 层先法/CFOP/其他
  cross_level:
    cross: 初级/中级/高级
    f2l: 初级/中级/高级
    oll: 初级/中级/高级
    pll: 初级/中级/高级
  formulas_learned:
    f2l: []
    oll: []
    pll: []

personal_bests:
  single: 999.99
  ao5: 999.99
  ao12: 999.99
  ao100: 999.99

milestones:
  - date: YYYY-MM-DD
    achievement: 成就描述
    time: 当时成绩

training_log: []
```

## 训练记录表头

每次训练会话记录格式：

```yaml
session_id: S20250506_001
date: 2025-05-06
user: 用户名
duration_minutes: 训练时长（分钟）
focus_area: [十字/F2L/OLL/PLL/全还原
solves:
  - solve_id: 1
    time: 45.23
    penalty: +2/DNF/OK
    scramble: 打乱公式
    notes: 备注
  - solve_id: 2
    time: 42.15
    penalty: OK
    notes:

statistics:
  count: 还原次数
  best: 最佳单次
  worst: 最差单次
  ao5: 五次平均
  ao12: 十二次平均
  mean: 平均时间
  std_dev: 标准差
```

## 预设用户水平分级标准

| 等级 | 名称 | AO5范围 | 特点 |
|------|------|----------|------|
| L1 | 新手 | >120s+ | 刚学会层先法 |
| L2 | 入门 | 60-120s | 熟练层先法 |
| L3 | 进阶 | 45-60s | 开始学CFOP |
| L4 | 熟练 | 30-45s | CFOP进行中 |
| L5 | 高手 | 20-30s | CFOP基本完成 |
| L6 | 精英 | 15-20s | 开始高级技巧 |
| L7 | 专家 | 12-15s | 全面优化 |
| L8 | 大师 | 10-12s | 准比赛级 |
| L9 | 顶级 | 8-10s | 国内比赛水平 |
| L10 | 世界 | <8s | 世界级水平 |
```
