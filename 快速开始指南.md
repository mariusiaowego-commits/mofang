# 快速开始指南

## 第一步：评估你的水平

回答以下问题，确定你在哪里：

### 🔍 快速评估
1. 你能独立还原魔方吗？ ❌ / ✅
2. 你用什么方法？ 层先法 / CFOP / 其他
3. 你大概需要多久？ ____ 秒/分钟

### 🎯 阶段对照表
| 你的情况 | 所属阶段 | 建议起点 |
|----------|----------|----------|
| 还不会还原 | L1 新手 | 入门层先法教程 |
| 能还原但 >2分钟 | L2 入门 | 优化层先法手法 |

| 20-30秒 | L5 高手 | 完整CFOP优化 |
| <20秒 | L6+ 精英 | 高级技巧 |

## 第三步：开始今天的训练

### 📅 第一天训练建议（30分钟）

根据你的水平选择：

#### 如果你是新手（还不会还原）
- 15分钟：学习层先法第一步和第二步
- 10分钟：反复练习这两步
- 5分钟：休息 + 回顾

#### 如果你能还原但 >2分钟
- 5分钟：十字专项练习（只做十字，20个）
- 20分钟：完整还原 10-15 次
- 5分钟：记录时间 + 分析问题

#### 如果你在 45秒-1分钟
- 5分钟：十字练习
- 10分钟：F2L配对练习
- 10分钟：完整还原
- 5分钟：学习1个新PLL公式

## 💡 学习小贴士

1. **不要急于求成**：稳扎稳打比盲目堆量重要
2. **肌肉记忆需要时间**：一个公式至少练50遍
3. **每天都练一会儿**：每天20分钟 > 周末一次2小时
4. **记录很重要**：看着数据进步更有动力
5. **瓶颈期很正常**：每个人都会遇到，坚持就会突破
