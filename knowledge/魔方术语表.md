# 魔方术语表

> [!note] 说明
> 收集整理魔方圈常用专业术语，从入门到进阶，一网打尽！

---

## 基础结构术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **中心块** | Center | 6个，固定不动，每个面的中心，决定了这个面的颜色 |
| **棱块** | Edge | 12个，每个有2种颜色，在两个面交界处 |
| **角块** | Corner | 8个，每个有3种颜色，在三个面交界处 |
| **层** | Layer | 魔方的一层，通常有三层（顶层、中层、底层） |
| **面** | Face | 魔方的一个面，共6个 |
| **色向** | Orientation | 块的颜色方向是否正确 |
| **位置** | Permutation | 块的位置是否正确 |

## 转动符号

| 符号 | 含义 | 符号 | 含义 |
|------|------|------|------|
| R | 右面顺时针90° | R' | 右面逆时针90° |
| L | 左面顺时针90° | L' | 左面逆时针90° |
| U | 顶面顺时针90° | U' | 顶面逆时针90° |
| D | 底面顺时针90° | D' | 底面逆时针90° |
| F | 前面顺时针90° | F' | 前面逆时针90° |
| B | 后面顺时针90° | B' | 后面逆时针90° |
| M | 中间层（RL之间） | E | 中间层（UD之间） |
| S | 中间层（FB之间） | x | 整体按R方向转 |
| y | 整体按U方向转 | z | 整体按F方向转 |
| w | 宽转（两层一起） | 2 | 转180°（如R2） |

> [!tip] 顺时针/逆时针判断
> **永远是面朝那个面来看！
> 这是新手最容易搞错的地方，记住这个原则就不会错！

## 还原方法相关

| 术语 | 英文 | 说明 |
|------|------|------|
| **层先法** | Layer by Layer / LBL | 入门方法，一层一层还原 |
| **CFOP** | Cross-F2L-OLL-PLL | 主流速拧方法，也叫Fridrich Method |
| **十字** | Cross | CFOP第一步，底面十字 |
| **F2L** | First Two Layers | 前两层同时还原 |
| **OLL** | Orientation of Last Layer | 最后一层翻色 |
| **PLL** | Permutation of Last Layer | 最后一层位置还原 |
| **COLL** | Corners of Last Layer | OLL变种，先角后棱 |
| **ZBLL** | Zborowski-Bruchem Last Layer | 一步还原最后一层（高手用） |
| **角先法** | Corners First | 先还原角块，后还原棱块 |
| **棱先法** | Edges First | 先还原棱块，后还原角块 |
| **桥式** | Roux Method | 另一种主流速拧方法 |
| **最小步** | Fewest Moves / FMC | 比赛项目，用最少步数还原 |

## 统计与成绩相关

| 术语 | 英文 | 说明 |
|------|------|------|
| **单次** | Single | 单次还原时间 |
| **PB** | Personal Best | 个人最佳成绩 |
| **PR** | Personal Record | 同PB，个人记录 |
| **AO5** | Average of 5 | 5次平均，去掉最快最慢 |
| **AO12** | Average of 12 | 12次平均，去掉最快最慢 |
| **AO100** | Average of 100 | 100次平均，去掉头尾各5个 |
| **Mean** | Mean | 简单平均，不去除极值 |
| **DNF** | Did Not Finish | 未完成还原（计为无效） |
| **DNS** | Did Not Start | 未开始 |
| **+2** | Plus Two | 惩罚，时间加2秒 |
| **WR** | World Record | 世界纪录 |
| **CR** | Continental Record | 大洲纪录 |
| **NR** | National Record | 国家纪录 |
| **Sub-X** | Sub X | 在X秒以内，如Sub10 = 10秒以内 |

> [!example] 举例
> - Sub30 = 稳定在30秒以内
> PB Sub15 = 个人最佳在15秒以内
> AO5 Sub20 = 五次平均在20秒以内

## 技巧与手法

| 术语 | 英文 | 说明 |
|------|------|------|
| **观察** | Inspection | 还原前15秒观察时间 |
| **预判** | Lookahead | 做当前步骤时观察下一步 |
| **停顿** | Pause | 步骤之间的停顿时间 |
| **TPS** | Turns Per Second | 每秒转动次数，手速指标 |
| **AUF** | Adjust U Face | 最后一层U面调整 |
| **入槽** | Slotting | F2L中把配对好的角棱放入槽位 |
| **藏角** | Corner Hide | F2L技巧，把角藏到底层 |
| **藏棱** | Edge Hide | F2L技巧，把棱藏到底层 |
| **非标** | Non-standard F2L | 非标准F2L情况 |
| **多向** | Multi-directional | 从多个方向做同一个公式 |
| **消步** | Move Cancel | 公式衔接时消除多余步骤 |
| **连贯** | Fluidity | 还原过程流畅无停顿 |
| **肌肉记忆** | Muscle Memory | 通过重复让手记住动作 |
| **条件反射** | Reflex | 看到情况手自动开始拧 |

> [!important] TPS参考值
> - 新手：2-3 TPS
> - 进阶：4-5 TPS
> - 高手：6-7 TPS
> - 世界级：8+ TPS

## 进阶概念

| 术语 | 英文 | 说明 |
|------|------|------|
| **颜色中立** | Color Neutral | 从任意颜色开始还原 |
| **十字中立** | Cross Neutral | 从任意颜色做十字 |
| **XCross** | Extended Cross | 十字 + 第一组F2L一起做 |
| **预判到OLL** | OLL Skip | 做完F2L直接OLL已经做好了 |
| **预判到PLL** | PLL Skip | 做完OLL直接PLL已经做好了 |
| **全预判** | Full Lookahead | 全程都在看下一步，几乎无停顿 |
