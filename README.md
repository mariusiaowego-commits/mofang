## 学习路线图

### 第一阶段：入门（目标：2分钟内）
- 掌握层先法（7步）
- 理解魔方基本术语
- 学会正确的握法和手法

### 第二阶段：进阶（目标：1分钟内）
- 学习CFOP基础
- Cross（十字）优化
- F2L理解和学习
- 开始肌肉记忆训练

### 第三阶段：熟练（目标：30秒内）
- 完整CFOP
- 7个十字OLL + 21个PLL
- 观察技巧提升
- AO5稳定

### 第四阶段：高手（目标：15秒内）
- 完整OLL（57个）
- F2L非标
- 预判和衔接
- 专业手法优化

### 第五阶段：顶级（目标：10秒内）
- COLL/ZBLL等高级技巧
- 多向公式
- 15秒观察规划
- 比赛心理训练

## 训练记录标准

- **单次时间（Single）**：单次还原时间
- **AO5（Average of 5）**：5次平均，去掉最快最慢
- **AO12（Average of 12）**：12次平均，去掉最快最慢
- **PB（Personal Best）**：个人最佳
