# 7个十字OLL 公式卡片

> [!success] 说明
> 当你已经做出黄色顶面十字时，只有这7种情况！
> **这7个是必须首先掌握的OLL公式，投入产出比最高！**

掌握这7个 + 21个PLL，你的CFOP就完成一大半了！

![[05-OLL-其他5个.png|600]]

## ✅ OLL 21 - Sune (小鱼1)

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 左后角 |
| **形状** | 小鱼形状，鱼头在左后方 |
| **公式** | `R U R' U R U2 R'` |
| **难度** | ⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

> [!tip] 手法提示
> 所有魔方玩家的"肌肉记忆"代表！
> 右手食指和中指交替拨U，极其流畅！

---

## ✅ OLL 22 - Anti-Sune (小鱼2)

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 右前角 |
| **形状** | 小鱼形状，鱼头在右前方（Sune的镜像） |
| **公式** | `L' U' L U' L' U2 L` |
| **难度** | ⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

> [!tip] 手法提示
> Sune的左手镜像版本！
> 左手食指和无名指配合。

---

## ✅ OLL 23

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 两个对角黄色 |
| **形状** | 点 + 对角两点 |
| **公式** | `R U2 R' U' R U' R'` |
| **难度** | ⭐⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

> [!tip] 手法提示
> 开头是U2，注意跟Sune区分！

---

## ✅ OLL 24

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 另外两个对角黄色 |
| **形状** | OLL 23的镜像 |
| **公式** | `L' U2 L U L' U L` |
| **难度** | ⭐⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

---

## ✅ OLL 25 - 三点钟

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 左后 + 右前 + 左前 |
| **形状** | 像箭头指向前方 |
| **公式** | `R U R' U R U' R' U R U2 R'` |
| **难度** | ⭐⭐ |
| **重要性** | ⭐⭐⭐⭐ |

> [!tip] 记忆技巧
> 前面跟Sune一样 R U R' U R，
> 然后多一个 U' R'，
> 最后再接 Sune！

---

## ✅ OLL 26 - 双角对边（弓）

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 左右两个角黄色，像一张弓 |
| **形状** | 横线双角 |
| **公式** | `f (R U R' U') (R U R' U') f'` |
| **难度** | ⭐⭐⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

> [!tip] 手法提示
> 两个Sune开头连做 + 宽f转！
> `f R U R' U' R U R' U' f'`

---

## ✅ OLL 27 - 十字（灯火）

| 项目 | 内容 |
|------|------|
| **黄色块位置** | 前后左右四个棱黄色 |
| **形状** | 十字形，中间是空的 |
| **公式** | `F (R U R' U') F'` |
| **难度** | ⭐⭐ |
| **重要性** | ⭐⭐⭐⭐⭐ |

> [!tip] 手法提示
> 就是做顶层十字的那个公式！
> 你早就会了对吧！😉
